# CS 1102 - Java Quiz Game Simulation

A console-based, interactive multiple-choice quiz game written in Java. This project demonstrates foundational programming concepts including input validation, logical branching, and type casting for precise computational tracking.

## Features
* **Five Academic Questions:** Covers essential programming topics with multiple-choice structures (A, B, C, D).
* **Robust Input Validation:** Traps incorrect inputs (such as numbers, empty spaces, or out-of-bounds letters) using conditional loops, requesting valid data before allowing the game loop to continue.
* **Smart Conditional Handling:** Integrates `if` statement logic for data constraint verification and `switch-case` blocks for clean, high-performance answer checking.
* **Precise Performance Summary:** Outputs metrics showing total correct answers alongside a calculated percentage score.

## How to Run the Program
1. Clone or download the repository files into your local directory.
2. Open your terminal or Command Prompt and navigate to the directory containing `QuizGame.java`.
3. Compile the Java source file into executable bytecode by running:
   ```bash
   javac QuizGame.java
